# React--To-Do-List-App
A simple and clean React To-Do List App built using Vite. Features include adding tasks, editing tasks, deleting tasks, and marking tasks as complete. A beginner-friendly project showcasing React state management and component design.

## Features
- Add, delete, and mark tasks as done
- Responsive design
- Keyboard & button input

## How to Run
1. Unzip the project
2. Open terminal in the folder
3. Run `npm install`
4. Run `npm run dev`
5. Open browser at http://localhost:5173
